## vue-advance
vue-advance 中的`vue2-multiple`是基于`webpack2.x`实现的多页打包功能模版。

## 使用
在使用 `vue-multiple` 模版之前必须先安装 `vue-clie`
```bash
npm install -g vue-cli
vue init zhaoqize/vue-advance my-project
```

运行模版
```bash
npm run dev
```

由于模版主要实现的是多页面打包配置，所以具体的业务写法与之前没有区别。
