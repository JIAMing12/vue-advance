## vue-advance
vue-advance 中的`vue2-multiple`是基于`webpack2.x`实现的多页打包功能模版。

## 使用
clone
```js
git clone https://github.com/zhaoqize/vue-advance.git
```

然后
```js
cd vue2-multiple
```

依照现有的打包模版根据具体项目进行修改即可。
